# Definiciones Técnicas ("Modern Delight")

## Stack Tecnológico

### Frontend
- **Framework:** Flutter (Multiplataforma).
- **Animaciones:** Rive (Animaciones interactivas ligeras).
- **Objetivo:** Experiencia de usuario fluida y visualmente rica ("Juice").

### Backend
- **Plataforma:** Supabase.
- **Base de Datos:** PostgreSQL (Manejo de data relacional).
- **Lógica:** Edge Functions (Escalabilidad elástica sin costo fijo alto).
- **Motor:** Reglas heurísticas implementadas en base de datos/funciones.

### Infraestructura y Aceleradores
- **Aceleradores (Interno):** Uso de UI Kits y AI (Cursor/Copilot).
- **Meta:** Reducir carga operativa de ~480h a ~350h.

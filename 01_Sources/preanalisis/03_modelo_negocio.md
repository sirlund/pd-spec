# Modelo de Negocio y Presupuesto

## Resumen Financiero
- **Tarifa Base:** $25 USD/hora (Senior).
- **Presupuesto Final (Rango):** $12,000 - $16,000 USD (Bruto).
- **Estructura Legal:** Boleta de Honorarios (Persona Natural) + Contrato de Cesión de IP.
    - *Nota:* Boleta exenta de IVA. Cliente retiene 15,5% (Tasa 2026).

## Estructura de Cobro por Hitos

### Hito 1: Prototipo y Fundamentos (30%)
- Prototipo Real en código (Flutter).
- Implementación de UI Kit.
- Diseño de Sonido inicial.

### Hito 2: Funcionalidad Alpha (40%)
- Alpha "Happy Path" funcional.
- Backend conectado (Supabase).
- Motor Heurístico implementado.

### Hito 3: Beta y Lanzamiento (30%)
- Beta final.
- Integración Rive ("Juice").
- Seguridad.
- Deploy.

## Estrategia de Margen
- El presupuesto permite subcontratar (~$2.5k-$3k) a un dev Backend/Fullstack para tareas de seguridad e infraestructura, liberando horas de Product Design/Frontend.

# Insights Graph

Atomic verified insights extracted from sources. Each insight has a unique `[IG-XX]` ID, a status, and a traceable source reference.

## Status Legend

- **PENDING** — Extracted but not yet verified or cross-referenced.
- **VERIFIED** — Confirmed through cross-referencing or user validation.
- **FROZEN** — Valid but deprioritized for current scope.
- **INVALIDATED** — Contradicted by stronger evidence; no longer active.
- **MERGED** — Combined with another insight during conflict resolution.
- **SUPERSEDED** — Replaced by a newer, more complete insight.

### [IG-SYNTH-02] Geometría del Plan Minero — Desalineación entre dueños y usuarios operativos
**Consolidates:** IG-15, IG-16, IG-17
**Convergence:** 1/1 sources
**Category:** user-need (current)
**Voice:** stakeholder
**Authority:** observation

> **Narrative:** El plan minero representa el plan de negocio del dueño, pero el dueño no es usuario del sistema. Los usuarios operativos se enfocan en ejecutar sus tareas sin visibilidad del impacto en el plan mayor, creando una brecha estructural entre estrategia y ejecución.

**Evidence trail:**
- "El plan minero está superalineado con el dueño - el plan minero es el plan de negocio"
- "El dueño no es usuario - ahí está el desafío"
- "Nuestros usuarios no están preocupados de la línea del dueño, están preocupados de hacer su pega"

**Named concept origin:** "Geometría" implied from geometric separation between stakeholders

### [IG-SYNTH-03] Arquitectura Tricapa — Integración, Inteligencia y Experiencia como stack tecnológico
**Consolidates:** IG-08, IG-09, IG-10, IG-11
**Convergence:** 1/1 sources
**Category:** technical (current)
**Voice:** stakeholder
**Authority:** fact

> **Narrative:** La plataforma se estructura en tres capas: (1) integración como fuente única de verdad para datos operativos, (2) inteligencia con topografía y sensores virtuales, (3) UX/UI donde solo se aprovecha "la punta del iceberg". La arquitectura está construida pero subutilizada.

**Evidence trail:**
- "La tecnología tiene tres capas: (1) integración de datos en tiempo real, (2) inteligencia (topografía, sensores virtuales), (3) UX/UI"
- "Capa de integración maneja diagnósticos en tiempo real - es la única fuente de la verdad"
- "Es como un iceberg - solo estamos sacándole provecho a la punta"

**Named concept origin:** "Tricapa" derived from three-layer architecture description
